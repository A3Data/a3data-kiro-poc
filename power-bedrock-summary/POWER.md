# Bedrock Summary Power

Este Power demonstra como o Kiro pode integrar com o Amazon Bedrock
para gerar resumos de texto com comportamento controlado.

## O que este Power faz
- Ativa uma integração com o Bedrock
- Aplica Steering de resumo simples
- Retorna respostas curtas e objetivas

Sempre que o usuário pedir:
- resumo de texto
- explicação curta
