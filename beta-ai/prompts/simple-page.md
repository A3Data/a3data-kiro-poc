Crie uma aplicação visual simples para validar uma ideia rapidamente.

Objetivo do projeto:
Demonstrar um fluxo básico de uso de um sistema, de forma clara e funcional, sem complexidade técnica.

O que essa aplicação precisa mostrar:
- Uma tela única
- Um formulário simples para cadastrar informações
- Uma lista com os dados cadastrados
- Possibilidade de editar e remover itens
- Tudo funcionando apenas para demonstração

Regras importantes:
- A aplicação deve ser simples e fácil de entender
- Foco em experiência visual e fluxo de uso
- Não precisa de integrações externas
- Não precisa salvar dados
- Não precisa de login ou permissões
- Não precisa de múltiplas telas

Comportamento esperado:
- O usuário preenche um formulário
- Os dados aparecem imediatamente na tela
- O usuário consegue editar ou excluir os dados
- Tudo acontece de forma rápida e intuitiva

Visual:
- Interface moderna, limpa e profissional
- Layout organizado e centralizado
- Design que passe sensação de produto real
- Utilizar as cores padrão do Kiro
- Evitar poluição visual ou excesso de elementos

Entrega:
- Gere tudo pronto para rodar
- A aplicação deve abrir e funcionar com poucos comandos
- Não explique decisões técnicas
- Apenas entregue o resultado funcional

O foco é demonstrar valor, não tecnologia.
